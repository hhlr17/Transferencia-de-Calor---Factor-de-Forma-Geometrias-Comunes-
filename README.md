Programa para TI CX (CAS Y NO CAS, VERSIÓN I Y VERSION II) que te ayuda a calcular el factor de forma, factor de visión o factor geometrico para algunas configuraciones geometricas de tamaño finito que se encuentran en la tabla 13-1 del libro "Transferencia de Calor y Masa, 4Ed, Cengel"

paralelasplanas(x,y,l), correspondiente a placas planas paralelas concentricas(rectangulares o cuadradas)
perpendicular(x,y,z), plascas planas perpendiculares con una arista en comun(rectangulares o cuadradas)
discparalecoax(ri,rj,l), discos paralelos coaxiales (mismos o diferentes radios)
